# Connector Auth Sender

## Установка
1. Откройте `chrome://extensions` или `edge://extensions`.
2. Включите **Режим разработчика**.
3. Нажмите **Загрузить распакованное** и выберите папку `connector_auth_sender`.

## Использование
1. Откройте сайт поставщика и выполните вход (капча/2FA).
2. Нажмите на иконку расширения.
3. Укажите адрес системы (например, `https://cargo.example.com`) и ID коннектора.
4. Нажмите **Собрать и отправить** — токен/куки будут отправлены в систему.

## Важно
- Оператор должен быть авторизован в системе в том же браузере, чтобы запрос прошёл.
- Расширение берёт токен из `LocalStorage/SessionStorage` по ключам, указанным в поле.
